If you experience problems with the applet try the following:
  1. Reload the page
  2. Super-Reload the page: Click Reload while holding down the Option key
  3. Clear the Memory/Disk Cache (using the Browser, or manually)
  4. Set the Disk Cache to 0 Megs
  5. Select Checks Documents: Every Time
  6. Disable Persistent Caching of Pages Retrieved Through SSL
  7. Quit the Browser and Restart
  8. File A Bug Report
  9. Pray for Divine Intervention

    If you experience problems with java on a Macintosh (specifically 68k Macs running System 7.6) try using System 7.5.5 or earlier till system System 7.6.1 is released.

    If you experience problems under Unix (eg SunOS 5.5x) it may be related to your $CLASSPATH so, try unsetting it or have it include the .netscape/java_30x files.
