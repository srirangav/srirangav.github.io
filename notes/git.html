Working with Branches

List local branches:
$ git branch
List remote branches:
$ git branch -r
List all branches:
$ git branch -a
List current branch:
$ git rev-parse --abbrev-ref HEAD
Delete local branch:

$ git branch -D branch
where branch is the name of the branch to be deleted.

Delete remote branch:

$ git push origin --delete branch
where branch is the name of the branch to be deleted.

Synchronize local branches with remote branches:
$ git fetch -p
Rename local branches:

$ git checkout master
$ git branch -m oldname newname
where oldname is the existing name for the branch and newname is the new name for the branch.

Rebasing Commits

  1. To rebase a commit, count the number of commits from the first commit to the last commit that you want to rebase and run git rebase as follows: 

    $ git rebase -i HEAD~num
    where num is the number of commits from the first commit through the last commit to be rebased, inclusive.

  2. In the editor that opens, mark the first commit as pick, and the remaining as squash.
See: https://www.git-tower.com/learn/git/faq/git-squash/
