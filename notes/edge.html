Edge Notes

The these are some general notes about Microsoft Edge.

  • Settings
  • Addons

Settings

Show Full Web Addresses: Select the menu "View" and the submenu "Always show full URLs"
Page Settings
  1. Click on "Page Settings" (Gear icon) on the start page
  2. Select Custom
  3. Disable "quick links", "Background" ("Off"), and "Conent" ("Content Off")
  4. Go Back to Page Settings
  5. Select "Advanced"
  6. Disable "New tab tips"
Preferences
  1. Select the Menu "Microsoft Edge" and the submenu "Preferences"
  2. Select Profiles in the settings sidebar
  3. Select "Personal info"
  4. Disable "Save and fill personal info"
  5. Go back to Profiles
  6. Select "Passwords"
  7. Disable "Offer to save passwords"
  8. Disable "Sign in automatically"
  9. Go back to Profiles
  10. Select "Payment info"
  11. Disable "Save and fill payment info"
  12. Go back to Profiles
  13. Select "Privacy, Search, and Services"
  14. Select "Strict" for "Tracking prevention"
  15. Select "Clear browsing data on close"
  16. Enable all options
  17. Go back to "Privacy, Search, and Services"
  18. Enable "Send 'Do Not Track' requests"
  19. Disable "Allow sites to check if you have payment methods saved"
  20. Enable "Block potentially unwanted apps"
  21. Disable "Use a web service to help resolve navigation errors", "Suggest similar sites when a website can't be found", "Show suggestions from Pinterest in Collections", "Save time and money with Shopping in Microsoft Edge"
  22. Select "Address bar and search"
  23. Disable "Show me search and site suggestions using my typed characters", "Show me suggestions from history, favorites and other data on this device using my typed characters"
  24. Change "Search engine used in the address bar" to "DuckDuckGo"
  25. Select "Manage search engines" and delete all search engines except for "DuckDuckGo"
  26. Select "New Tab Page" in the Settings sidebar
  27. Disable "Preload the new tab page for a faster experience"
  28. Select "Cookies and data stored" in the Settings sidebar
  29. Select "Manage and delete cookies and site data"
  30. Enable "Block third-party cookies"
  31. Disable "Preload pages for faster browsing and searching"
  32. Go back to "Site permissions"
  33. Select "Location"
  34. Disable "Ask before accessing (recommended)"
  35. Go back to "Site permissions"
  36. Select "Motion or light sensors"
  37. Disable "Allowed (Recommended)"
  38. Go back to "Site permissions"
  39. Select "Notifciations"
  40. Disable "Ask before sending (recommended)"
  41. Go back to "Site permissions"
  42. Select "Background sync"
  43. Disable "Allow recently closed sites to finish sending and receiving data (recommended)"
  44. Go back to "Site permissions"
  45. Select "Automatic downloads"
  46. Disable "Ask when a site tries to download multiple files automatically (recommended)"
  47. Go back to "Site permissions"
  48. Select "MIDI devices"
  49. Disable "Ask when a site wants to use system exclusive messages to access MIDI devices (recommended)"
  50. Go back to "Site permissions"
  51. Select "USB devices"
  52. Disable "Ask when a site wants to access USB devices (recommended)"
  53. Go back to "Site permissions"
  54. Select "Serial ports"
  55. Disable "Ask when a site wants to access serial ports (recommended)"
  56. Go back to "Site permissions"
  57. Select "File editing"
  58. Disable "Ask when a site wants to edit files or folders on your device (recommended)"
  59. Go back to "Site permissions"
  60. Select "PDF Documents:
  61. Enable "Always download PDF files"
  62. Go back to "Site permissions"
  63. Select "Clipboard"
  64. Disable "Ask when a site wants to see text and images copied to the clipboard (recommended)"
  65. Go back to "Site permissions"
  66. Select "Payment handlers"
  67. Disable "Allow sites to install payment handlers (recommended)"
  68. Go back to "Site permissions"
  69. Select "Media autopay"
  70. Select "Limit"
  71. Go back to "Site permissions"
  72. Select "Virtual reality"
  73. Disable "Ask when a site wants to use your virtual reality devices and data (recommended)"
  74. Go back to "Site permissions"
  75. Select "Augmented reality"
  76. Disable "Ask when a site wants to create a 3D map of your surroundings or track camera position (recommended)"
