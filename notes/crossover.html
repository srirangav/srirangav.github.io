Running a Windows Program Directly
----------------------------------

1. Create a "New Bottle" in the "Bottle" menu

2. Select "Run Command" once the bottle has been created

3. In the "Run Command" window, select "Browse" and find the
   Windows program to run

4. (Optional) Click "Save Command as a Launcher" to have an
   launch icon created for the command

5. Click "Run" in the "Run Command" window to run the program

From: https://www.codeweavers.com/support/wiki/mac/mactutorial/run_only

Running / Installing Particular Windows Programs
------------------------------------------------

1. Running The Captain:

   a. Use a Windows 8 64-bit Bottle

   b. Use the GoG installer

   c. Add the optional CrossOver HTML engine and Direct X for Modern
      Games components

2. Running The Librarian

   a. Use a Windows 10 64-bit Bottle

   b. Use the GoG installer

   c. Add the optional CrossOver HTML engine and Direct X for Modern
      Games components

3. Running NiBiRu: Age of Secrets

   a. Use as Windows XP 32-bit Bottle

   b. Use the installer from the DVD/CD

   c. Add the following optional components:

      i.   .Net Framework 2.0, 3.0, 3.

      ii.  CrossOver HTML engine (IE 8.0)

      iii. Direct X for Pre-Windows XP Games

4. Running Murder on the Orient Express (and other Adventure Company Agatha
   Christie games):

   a. Use a Windows 2000 Bottle

   b. Use the installer from the CD/DVD

   c. Add the following optional components:

      i.  CrossOver HTML engine (IE 8.0)

      ii. Direct X for Pre-Windows XP Games

5. Running Chronicles of Mystery Scorpio Ritual and Tree of Life:

   a. Use a Windows XP 32-bit Bottle

   b. Use the installer from the CD/DVD

   c. Add the following optional components:

      i.   CrossOver HTML engine (IE 8.0)

      ii.  Microsoft Visual C++ 2005 Redistributable

      iii. Direct X for Pre-Windows XP Games

6. Running Hercule Poirot: The London Case

   a. Use a Windows 10 64-bit Bottle

   b. Use the GoG installer

   c. Add the optional CrossOver HTML engine and Direct X for Modern
      Games components
