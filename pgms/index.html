MacOS X Programs
----------------

 1. Browser Ram Cache - a perl script that creates a RAM
    disk for Chrome, Opera, and Safari disk caches:

    https://github.com/srirangav/BrowserRamCache

 2. Clean Chrome - a perl script that removes old versions
    of Chrome:

    https://github.com/srirangav/clean_chrome

 3. displayutil - a command line utility for retrieving
    information about, enabling, or disabling darkmode,
    grayscale, and night shift:

    https://github.com/srirangav/displayutil

 4. Hash - an application to calculate and verify hashes
    for files:

    https://github.com/srirangav/Hash

 5. Grayscale - an menu bar item to toggle the display
    between color and grayscale:

    https://github.com/srirangav/Grayscale

 6. MenuCal - a menu bar calendar:

    https://github.com/srirangav/MenuCal

 7. pdfview - a command line tool to display the text
    in a pdf:

    https://github.com/srirangav/pdfview

 8. QuickLook Clear Cache - a login item that clears the
    QuickLook cache every two hours:

    https://github.com/srirangav/qlClearCache

 9. QuickLook Image Preview With Size - a QuickLook plugin
    that includes image size and dimensions in the title bar:

    https://github.com/srirangav/qlImagePreviewWithSize

10. QuickLook Zip File Info — a QuickLook plugin that
    lists the contents of various archive files:

    https://github.com/srirangav/qlZipInfo

11. SimpleClockSaver - a simple digital clock screensaver:

    https://github.com/srirangav/SimpleClockSaver

12. Unhide Library - a login item that makes ~/Library visible:

    https://github.com/srirangav/unhidelib

13. vocr - a command line utility for ocr'ing images and pdfs:

     https://github.com/srirangav/vocr

14. Webloc to Url (wl2url) - a perl script that converts .webloc
    files to .url files:

    https://github.com/srirangav/wl2url
